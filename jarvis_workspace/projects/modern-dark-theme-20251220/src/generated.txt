# CRM Dashboard

A modern CRM with a dark theme using React.

## Features

- Contacts
- Deals
- Pipeline
- Notes
- Search

## Installation

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the server: `npm start`

## Usage

The application is built with React and styled with CSS. It includes a dark theme and is designed to be production-ready.

## Technologies

- React
- CSS
- Babel
